# Personal Website

This project is a responsive personal website built to showcase my background, projects, and technical skills.  
It was developed as part of a university course and focuses on clean UI design, accessibility, and modern frontend best practices.

## Live Site
[View the website here](https://people.tamu.edu/~shashchath13/project1-shashchath13/)

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Features
- Responsive design for desktop
- Clear navigation across sections (About, Projects, Experience, Contact, AI Page)
- Modular and maintainable frontend structure
- Lightweight and fast-loading pages
- Light and Dark modes

## What I Learned
- Structuring a multi-section frontend project from scratch
- Writing clean, reusable CSS and JavaScript
- Building responsive layouts using modern CSS techniques
- Deploying and maintaining a live web application

## Project Context
This website was originally developed as part of a class project at Texas A&M University and has since been refined to better reflect my personal portfolio and technical interests.
