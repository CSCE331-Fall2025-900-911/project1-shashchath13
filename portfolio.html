Food Truck Project

In my CSCE 314 course, I developed a Cheese Sandwich Food Truck simulator using JavaFX and SceneBuilder. This was an individual project, requiring careful time management and iterative development to complete each component of the application. The project was structured so that every stage of the food truck process corresponded to a dedicated Java and FXML file, which I then integrated to allow seamless navigation between views. The final product, shown in the image on the right, includes both a customer-facing interface and an admin panel for managing inventory. Access to the administrative side is secured with a password prompt, and it provides functionality to restock supplies and monitor operations.

Technologies: Java, JavaFX, FXML, CSS, SceneBuilder
